# Guides

Explore guides covering common use cases and best practices for working with mongo.do.

## Available Guides

### [CRUD Operations](/docs/guides/crud)
Learn how to create, read, update, and delete documents in mongo.do. Covers insertOne, insertMany, find, findOne, updateOne, updateMany, deleteOne, and deleteMany operations.

### [Aggregation Pipelines](/docs/guides/aggregation)
Master MongoDB's powerful aggregation framework. Learn about $match, $group, $project, $sort, and other pipeline stages for complex data transformations.

### [Indexing](/docs/guides/indexing)
Optimize query performance with indexes. Learn how to create, manage, and use indexes effectively.

### [Transactions](/docs/guides/transactions)
Understand how to use multi-document transactions for data consistency and atomic operations.

### [MongoClient & Connection](/docs/guides/connection)
Connect to mongo.do and manage database connections. Learn about URI and Env modes, connection lifecycle, database selection, and session management.

### [Query & Filter Syntax](/docs/guides/queries)
Complete guide to mongo.do query operators and filter syntax. Covers comparison operators ($eq, $ne, $gt, $lt, $in, $nin), logical operators ($and, $or, $not, $nor), element operators ($exists, $type), array operators ($elemMatch, $size, $all), and evaluation operators ($regex, $mod).
